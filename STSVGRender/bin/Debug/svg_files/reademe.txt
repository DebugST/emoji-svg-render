Put all Emoji-svg files in this folder.
And the demo is uses Twoemoji:
https://github.com/twitter/twemoji/tree/master/assets/svg

The filename format:
    u?CodePoint(-CodePoint)*.svg
    u?CodePoint(_CodePoint)*.svg
e.g:
    1f60a.svg
    u1f60a.svg
    1f468-200d-1f384.svg
    u1f468-200d-1f384.svg

If you think the svg file is too big, you can use this project:
https://github.com/RazrFalcon/svgcleaner-gui
